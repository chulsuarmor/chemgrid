### [2026-02-28] VSEPR 각도 배분 알고리즘 v3.0

**문제:** v2.0에서 `base_angle + offset` 방식으로 H/LP를 배치하면, 기존 결합과 겹칠 수 있음.

**해결:** 빈 각도 구간(gap) 탐색 알고리즘으로 교체
1. 기존 결합 각도를 [0°, 360°) 정규화 후 정렬
2. 인접 결합 사이의 gap 크기 계산
3. 가장 큰 gap부터 확장 요소 배분 (gap 내 균등 분할)
4. VSEPR 이상 간격(`360°/total_groups`)에 따라 fit 개수 결정

**검증 결과 (6개 케이스):**
- H₂O (2결합+2LP): 193°, 276.5° — 기존 결합(0°, 109.5°)과 미겹침 ✅
- NH₃ (3결합+1LP): 60° — 기존 결합(0°, 120°, 240°) 사이 정중앙 ✅
- CH₃- (1결합+3H): 90°, 180°, 270° — 0° 반대편 균등 ✅
- C=O (1결합+2LP): 120°, 240° — 삼각평면 ✅
- 독립원자 (0결합+4LP): -90°, 0°, 90°, 180° — 균등 ✅
- HCl (1결합+3LP): 270°, 0°, 90° — 180° 반대편 균등 ✅

### [2026-03-01] 🔴 U5: 고리 내 이중결합 짧은 선 안쪽 방향 수정

**문제:** 벤젠 등 고리 분자에서 이중결합의 짧은 보조선(perp)이 항상 `+perp` 방향으로 그려짐.
고리 바깥쪽으로 향하는 경우가 발생 → ChemDraw 표준과 불일치.

**해결:** 고리 감지 + 중심 방향 정렬 알고리즘 추가
1. `_find_ring_containing_bond(k1, k2, adj)` — BFS로 k1→k2 우회 최단경로 탐색 (최대 8원자 고리)
2. `_get_ring_center_direction(k1, k2, analysis)` — 고리 중심점 계산 → 결합 중점에서 중심으로의 단위벡터 반환
3. `perp`와 `ring_dir`의 dot product로 방향 판정:
   - `dot >= 0` → perp가 이미 안쪽 방향 (유지)
   - `dot < 0` → perp 반전 (`-perp`)
4. 고리가 아닌 결합 → `ring_dir = None` → 기존 로직 유지

**적용 위치:**
- `LewisRenderer._render_bonds()` STAGE 1
- `TheoryRenderer.render()` STAGE 1 (Manager 긴급 지시로 수정)

**검증 결과 (5건 전부 통과):**
- 벤젠 6원 고리 감지 ✅
- 벤젠 고리 중심 방향 벡터 (dot=1.000, 완벽한 안쪽 방향) ✅
- 비고리 C=C → None 반환 ✅
- 피리딘 (N 포함 6원 고리) 감지 ✅
- 사이클로프로판 (3원 고리) 감지 ✅

**주의사항:**
- 나프탈렌 등 다환 고리: BFS가 최소 고리를 반환하므로 자동으로 올바른 중심 계산
- 최대 탐색 깊이 8로 제한하여 성능 보장
- 비고리 이중결합은 기존 로직 완전 유지 (영향 없음)
