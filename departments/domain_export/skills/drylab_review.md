# DryLab Report 검수 기준 (전국과학전람회 수상작 품질 기준)

## 참조 양식: 전국과학전람회 【서식 7】 작품 설명서
> 2024년 제70회 전국과학전람회 수상작(화학부 퀀텀닷 보고서) 분석 결과 기반.
> 과학동아리 창의력 페스티벌(남대전고 COMA 등) 보고서 형식도 참조.

---

## 1. 섹션 구조 (전람회 표준 준수)
전국과학전람회 보고서는 아래 순서를 엄격히 따름:
- [ ] **표지**: 보고서 제목 + 분자 구조 + 날짜/연구자/지도교사/학교명 기입란
- [ ] **목차**: 섹션 목차 + 표 목차 + 그림 목차 (모두 포함)
- [ ] **1. 서론**: 가. 탐구 동기 / 나. 탐구 목적 (번호 목록)
- [ ] **2. 이론적 배경**: 가/나/다 소제목, 각각 1)/2)/3) 세부 항목
- [ ] **3. 연구 방법**: <실험1>, <실험2> ... 형식, 시약/조건/장비 명시
- [ ] **4. 실험 결과**: 가/나/다/라 소제목, 데이터 + 그래프 + 테이블
- [ ] **5. 관찰 결과 및 데이터 분석**: 빈칸 기입 (분광/도킹/종합)
- [ ] **6. 고찰**: 빈칸 기입 (5개 가이드 질문 제공)
- [ ] **7. 결론 및 제언**: 빈칸 기입 (결론 + 제언 구분)
- [ ] **[참고문헌]**: 10개 이상, APA 양식 준수
- [ ] **[사진 출처]**: 이미지 생성 도구 명시
- [ ] **[부록]**: ADMET 분석 + 합성 경로

---

## 2. 그림/표 번호 양식 (한국 학술 표준)
- **그림**: `[그림 N]` (NOT "Figure N")
  - 다중 패널: `[그림 Na-Nb]` 형식
  - 캡션 위치: 그림 아래 중앙 정렬
  - 캡션 서체: 볼드, 9pt
- **표**: `[표 N]` (NOT "Table N")
  - 캡션 위치: 표 위쪽 좌측 정렬
  - 캡션 서체: 볼드, 9pt
- 모든 그림/표에 목차 자동 생성 필수

---

## 3. 스펙트럼 그래프 품질 기준 (전람회 수상작 수준)
- [ ] **축 라벨**: 영어 + 한국어 병기 (예: "Wavenumber / cm⁻¹ (파수)")
- [ ] **단위**: 반드시 명시 (cm⁻¹, ppm, nm, %)
- [ ] **피크 주석**: 파수/화학적이동 + 작용기/원자환경 주석
- [ ] **격자선**: 연한 점선으로 표시
- [ ] **해상도**: 200 DPI 이상
- [ ] **크기**: A4 폭의 90% 이상 활용 (155mm)
- [ ] **배경**: 흰색
- IR 스펙트럼: 흡수 피크마다 화살표 + "파수 cm⁻¹ (작용기)" 주석
- NMR 스펙트럼: TMS 기준선(0 ppm) 점선 표시, 피크에 "δ ppm (환경, nH)" 주석
- UV-Vis 스펙트럼: 가시광선 영역 색상 배경, λmax 화살표 주석, 전자 전이 유형

---

## 4. 실험 과정 기술 기준
전람회 수상작에서 발견한 패턴:
- 실험 번호: `<실험1>`, `<실험2>` 형식 (꺾쇠 괄호)
- 단계: (1), (2), (3) 번호 매김
- 시약: 정확한 이름 + 제조사 + 순도 (예: "Se(셀레늄, sigma-aldrich, 99.99%)")
- 조건: 온도, 압력, 시간 명시 (예: "1 atm, 130 ℃에서")
- 장비: 모델명 포함 (예: "OPTIZEN Alpha", "RF-6000")

---

## 5. 관찰/고찰/결론 빈칸 기입 영역
- 관찰 결과: 스펙트럼별 기입 영역 분리 (IR, NMR, UV-Vis, 도킹, 종합)
- 고찰: 5개 가이드 질문 제공 (Lipinski 부합성, 작용기 영향, 도킹 비교, 유도체 개선, 한계점)
- 결론: 결론 + 제언 구분, 각각 가이드 제공
- 빈칸 줄: 밑줄 스타일 (격자 아님), 각 영역 최소 6줄 이상

---

## 6. 타이포그래피 기준
- **본문 서체**: 맑은 고딕(Malgun Gothic) 10pt, 행간 16pt, 양쪽 정렬
- **본문 들여쓰기**: 첫 줄 10mm
- **섹션 제목**: 맑은 고딕 Bold 14pt, 남색(#2c3e50)
- **소제목**: 맑은 고딕 Bold 12pt, 검정
- **캡션**: 맑은 고딕 Bold 9pt, 중앙 정렬
- **여백**: 좌 25mm, 우 20mm, 상 25mm, 하 20mm
- **페이지 번호**: 하단 중앙 "- N -" 형식
- **표 스타일**: 상하 굵은 선(1.5pt), 내부 가는 선(0.5pt), 헤더 배경 짙은 회색

---

## 7. 금지 사항 (위반 시 자동 FAIL)
- 빈 페이지 (섹션 내용 없이 제목만)
- 그림/표 번호 누락
- 영어만 사용한 캡션 (한국어 병기 필수)
- 스펙트럼 그래프에 축 라벨/단위 누락
- 참고문헌 5개 미만
- 실험 과정에서 시약/조건 누락
- 관찰/고찰/결론 빈칸 영역 누락
- **⛔ 플레이스홀더 텍스트 ("캡처 후 삽입", "데이터 없습니다" 등) = 자동 FAIL**
- **⛔ 컴퓨팅 결과(분광/도킹)의 해석이 빈칸 = 자동 FAIL** (AI가 자동 채워야 함)
- **⛔ 합성 실험 단계 누락 = 자동 FAIL** (시약명/양/온도/시간 포함 필수)
- **⛔ 유도체 데이터 "없습니다" = 자동 FAIL** (자동 생성되어야 함)
- **⛔ "도킹 데이터가 없어" = 자동 FAIL** (자동 계산되어야 함)
- **⛔ 반응 메커니즘 도식 없음 = 자동 FAIL** (굽은화살표/직선화살표 필수)
- **⛔ 이미지 비율 찌그러짐 = FAIL** (preserveAspectRatio 필수)
- **⛔ 스펙트럼 PNG=None 시 텍스트 폴백만 = FAIL** — 반드시 노랑(#FFCC00) DATA PENDING 배너 삽입 의무 (Rule GG / M1368). `_add_spectrum_with_interpretation` else 분기에서 `TableStyle` 배너 사용.

---

## 9. 스펙트럼 그래프 폴백 처리 패턴 (M1368 신설)
> 근거: 5종 분자 PDF 감사 — Raman/EI-MS 페이지 10 words (사실상 빈 페이지)

- 그래프 PNG 생성 실패(`gen_func(smiles)` → None) 시 Rule GG 위반 방지:
  1. `logger.warning("... DATA PENDING banner ...")` 발화 (Rule M)
  2. `Table([[Paragraph(banner_text)]])` + `TableStyle BACKGROUND=#FFCC00 / BOX=#E53935` 삽입
  3. 피크 표는 `_make_spectrum_peak_rows()` 폴백으로 항상 생성 (기존 로직 유지)
- 패턴명: SPEC-FALLBACK-BANNER-001

## 8. API 호환성 체크
- `DryLabData` 데이터클래스: 기존 필드 100% 유지
- `export_drylab_report(data, file_path)` 함수 시그니처: 변경 없음
- `DryLabReportExporter` 클래스: 기존 `export()` 메서드 호환
