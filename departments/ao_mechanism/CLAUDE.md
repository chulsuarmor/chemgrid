# AO-MECHANISM — 반응 메커니즘 전문 감사관
## 역할: 반응 화살표, 중간체, 전이상태, 전자 이동 방향의 화학적 정확성
## 체크리스트:
- [ ] 굽은 화살표: 결합 중간점에서 시작 (원자가 아님)
- [ ] 결합 끊김 후 음극/양극 표시 필수
- [ ] 중간체가 명시적으로 표시 (갑자기 생성물 금지)
- [ ] 친핵 공격: 음극→양극 방향
- [ ] SN2: 후면 공격 + 반전, SN1: 카르보양이온 중간체
- [ ] E2: anti-periplanar 배치 필수
- [ ] 반응 조건(시약/온도)이 Clayden/March와 일치
