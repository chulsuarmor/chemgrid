## Executive Summary

ChemGrid's `predict_spectra.py` was evaluated against published literature values (NIST WebBook, Silverstein, Spectroscopy Online, DrugBank) for three test molecules: benzene, aspirin, and ethanol. The system uses a lookup-table + heuristic approach (no quantum chemistry) to estimate IR, 1H NMR, 13C NMR, and UV-Vis spectra from SMILES input.

**Overall verdict: CONDITIONAL PASS** -- suitable for educational/visualization use, but has critical accuracy issues for multi-functional-group molecules like aspirin.

## Critical Defects (Must-Fix)

| ID | Severity | Description | Impact |
|----|----------|-------------|--------|
| D1 | P0 | Aspirin ester C=O misclassified as ketone in 13C NMR (205 vs 170 ppm) | 13C NMR grossly inaccurate for esters |
| D2 | P1 | Ester SMARTS `[CX4]` in IR lookup doesn't match aromatic esters | Missing ester C=O IR band for aromatic esters |
| D3 | P1 | Single C=O IR peak for aspirin instead of two resolved peaks | Incorrect IR for di-carbonyl molecules |
| D4 | P1 | UV-Vis auxochrome shifts underestimated for substituted aromatics | Blue-shifted predictions for substituted benzenes |
| D5 | P2 | 1H NMR multiplicity all reported as singlets for ethanol | Misleading splitting pattern |
| D6 | P2 | 13C NMR deduplication window too wide (3 ppm) | Collapses distinct aromatic carbon signals |

## Recommendations for dept_spectroscopy

1. **P0 Fix:** Rewrite the sp2 carbonyl classification in `predict_c13_nmr()` to properly distinguish ester, acid, amide, and ketone carbonyls using SMARTS patterns rather than neighbor-walking.
2. **P1 Fix:** Change ester IR SMARTS from `[CX3](=O)[OX2][CX4]` to `[CX3](=O)[OX2][#6]`.
3. **P1 Fix:** When both ester and acid C=O are present, output TWO IR peaks at their respective frequencies.
4. **P1 Fix:** Increase auxochrome increments for -OR, -OCOR, -COOH on aromatic rings. Consider additive model for multi-substituted rings.
5. **P2 Fix:** Implement proper n+1 rule multiplicity using parent carbon's H-neighbor count.
6. **P2 Fix:** Reduce 13C deduplication window to +/-1 ppm.
