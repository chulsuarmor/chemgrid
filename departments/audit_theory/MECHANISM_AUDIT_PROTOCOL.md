# 반응 메커니즘 PDF 감사 프로토콜

## 목적
`mechanism_pdf_exporter.py`로 생성된 합성 경로 PDF의 화학적 정확성을 교과서/논문 기반으로 검증.

## 검증 항목

### 1. 반응 조건 정확성
- [ ] 시약/용매가 해당 반응에 적합한가? (예: Suzuki 커플링 → Pd 촉매 + 염기 필수)
- [ ] 온도/압력 조건이 현실적인가? (예: Grignard → 무수 조건, 0°C)
- [ ] 반응 시간이 합리적인가?

### 2. 반응 화살표 방향
- [ ] 굽은 화살표가 전자 이동 방향을 올바르게 표시하는가?
- [ ] 친핵체 → 친전자체 방향이 맞는가?
- [ ] 산-염기 반응에서 양성자 이동이 올바른가?

### 3. 중간체 구조
- [ ] 각 단계의 생성물이 화학적으로 유효한가?
- [ ] 입체화학이 유지/반전되는가? (SN2 → 반전, SN1 → 라세미)
- [ ] 전하 균형이 맞는가?

### 4. 교과서 대조
| 반응 유형 | 참고 교과서 | 대조 항목 |
|-----------|-----------|----------|
| 친핵치환 (SN1/SN2) | Clayden Ch.17 | 속도론, 입체 |
| 친전자방향족치환 | Clayden Ch.22 | 배향성, 활성화 |
| 알돌 축합 | Clayden Ch.27 | 교차알돌 선택성 |
| Suzuki 커플링 | Organic Reactions Vol.63 | Pd 산화 상태 |
| Grignard | March Ch.10 | 무수 조건, 부반응 |

### 5. 감사 절차
1. 감사 대상 분자 목록 (CT에서 지정)
2. 각 분자의 합성 경로 PDF 생성
3. R1 (구조화학): 중간체 구조 + 입체화학 검증
4. R2 (분광약리): 반응 조건 + 메커니즘 화살표 검증
5. TL: 교과서 대조 교차확인 → 서명
6. 불일치 → 구체적 수정 지시 → 코드 수정 → 재검증

## 출력물
- `evidence/mechanism_audit_YYYYMMDD.md`: 비교표
- `evidence/mechanism_pdfs/`: 감사 대상 PDF 모음
