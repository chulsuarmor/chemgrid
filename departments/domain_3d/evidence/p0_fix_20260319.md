# P0 Fix Evidence: AlphaFold/Drug-design Methods in Wrong Class

## Date: 2026-03-19
## Severity: P0 (AttributeError crash on 3D popup open)
## File: `src/app/popup_3d.py`

## Bug Description
6 methods (`_create_alphafold_synthesis_tab`, `_run_drug_design`, `_open_lead_optimizer`, `_open_alphafold`, `_open_synthesis`, `_open_admet`) were defined inside `DockingEnergyPanel` but called from `Molecule3DPopup._init_ui()`, causing `AttributeError`.

## Fix Applied
Moved all 6 methods from `DockingEnergyPanel` to `Molecule3DPopup` (inserted before `closeEvent`).
