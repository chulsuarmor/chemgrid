# 입체 구조 데이터 가이드 (3D Structure Asset Guide)

분광학 이미지 폴더(`spectra_assets/`)와 동일한 방식으로,
입체 구조 레이어의 두 가지 표현 방식에 대한 레퍼런스 이미지와 설명을 보관하는 공간입니다.

---

## 1. 폴더 구조

```
structure_assets/
├─ 일반_입체구조/          # 분자 입체구조(일반) 설명 및 이미지
│  ├─ 설명.txt
│  └─ (이미지 파일들...)
├─ 오비탈_표현구조/         # 분자 오비탈 표현 구조 설명 및 이미지
│  ├─ 설명.txt
│  └─ (이미지 파일들...)
└─ GUIDE.md                # 이 파일
```

---

## 2. 각 폴더의 목적

### 📁 `일반_입체구조/`

**대상**: 팝업의 "⚛ Ball & Stick" / "🔵 Space Filling" 모드 표현
- Ball-and-Stick: 원자(공)와 결합(막대)으로 입체 형태를 나타내는 방식
- Space-Filling(CPK): 반데르발스 반경 기준으로 원자 크기를 표현하는 방식

**여기에 넣을 수 있는 내용:**
- 정확한 3D 렌더링 레퍼런스 이미지 (PNG/JPG)
- 올바른 결합 길이/각도 기준값 (Å, °)
- VSEPR/결합각/분자 형태 설명 텍스트

**작성 양식 (설명.txt):**
```
[분자명]: 예) 벤젠, 물, 메탄
[렌더링 방식]: Ball-and-Stick / Space-Filling
[결합 길이]: C-C 1.40 Å (벤젠), O-H 0.96 Å (물) ...
[결합 각도]: H-C-H 109.5° (메탄), H-O-H 104.5° (물) ...
[분자 형태]: 정육면체형 / 삼각평면형 / 사면체형 ...
[레퍼런스 출처]: PubChem / NIST / 교과서 ...
```

---

### 📁 `오비탈_표현구조/`

**대상**: 팝업의 오비탈 모드 표현
- "🌀 π 오비탈 (sp2)": 벤젠/에틸렌 등 방향족 π cloud
- "⚗ 혼성 오비탈 (자동)": sp/sp2/sp3/sp3d/sp3d2 혼성 오비탈
- "⚛ d 오비탈 (전이금속)": t₂g/eg Crystal Field
- "✦ f 오비탈 (란타나이드)": f-block 원소 오비탈

**여기에 넣을 수 있는 내용:**
- ORCA/Avogadro에서 생성된 오비탈 이소서페이스 이미지 (PNG/JPG)
- 오비탈 형태의 이론적 설명 (각운동량 양자수, 위상 등)
- 각 혼성 유형별 기하 구조 설명

**작성 양식 (설명.txt):**
```
[오비탈 유형]: π / sp2 / sp3 / d(t2g) / d(eg) / f
[대상 분자/원소]: 예) 벤젠(π), 에틸렌(sp2), 메탄(sp3), Fe²⁺(d)
[이소밸류(isovalue)]: ORCA 기준 0.04 au (기본값)
[위상 표현]: 파랑=양의 위상, 빨강=음의 위상
[이론적 근거]: Hückel 규칙 / 결정장 이론 / ...
[ORCA 계산 설정]: ! B3LYP def2-TZVP NBO (필요 시)
```

---

## 3. 에이전트 인계 사항

- **06_3d_structure (입체구조 담당)** 가 이 폴더의 내용을 참조하여
  3D 팝업(`popup_3d.py`)의 렌더링 파라미터를 보정합니다.
- 이미지를 업로드한 후 `[분자명]_설명.txt` 형식으로 같은 폴더에 설명을 추가해 주세요.
- 분광학 `spectra_assets/`와 동일한 방식으로 AI 에이전트가 이 폴더를 분석하여
  렌더링 알고리즘에 반영합니다.

## 4. 사용법 예시

1. **일반 입체구조 레퍼런스 추가**:
   - `일반_입체구조/benzene_ball_stick.png` (벤젠 Ball-and-Stick 정답 이미지)
   - `일반_입체구조/benzene_설명.txt` (결합 길이 1.40Å, 결합각 120° 등)

2. **오비탈 표현 레퍼런스 추가**:
   - `오비탈_표현구조/benzene_pi_orbital.png` (ORCA isovalue 0.04 기준)
   - `오비탈_표현구조/benzene_pi_설명.txt` (π MO 형태, 위상 색상 기준 등)
