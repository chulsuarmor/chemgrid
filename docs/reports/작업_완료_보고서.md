# 고정된 1개 원자 허용 오차

**변경 후:**
```python
count_diff = max_count - min_count

# 동적 허용 오차: max(5% of max_count, 2 atoms)
dynamic_tolerance = max(
    int(max_count * ATOM_COUNT_TOLERANCE_PERCENT),
    ATOM_COUNT_TOLERANCE_MIN
)

if count_diff > dynamic_tolerance:
    print(f"\n  ⚠️  WARNING: Atom count mismatch exceeds tolerance")
    print(f"      Difference:     {count_diff} atoms")
    print(f"      Tolerance:      {dynamic_tolerance} atoms (5% or min 2)")
elif count_diff > 0:
    print(f"\n  ℹ️  INFO: Minor atom count difference (within tolerance)")
else:
    print(f"\n  ✓  All sections have consistent atom counts")
```

**핵심 개선사항:**
- ✅ 동적 허용 오차: 작은 분자(2개), 큰 분자(5% = 180원자 시 9개)
- ✅ 3단계 리포팅: Error / Info / Success
- ✅ 물리적 타당성: 파싱 오류를 5%까지 수용

### ✅ 원칙 2: 물리적 타당성 검증

**전하 허용 오차 근거:**
- DFT SCF 수렴 기준: $10^{-8}$ Hartree
- 부분전하 오차 전파: $10^{-5}$ ~ $10^{-6}$ 전자
- 안전 마진: $10^{-4}$ (대형 분자 누적 오차 고려)

**원자 개수 허용 오차 근거:**
- 파싱 오류: ORCA 출력의 다중 섹션에서 1-2개 원자 누락 가능
- 동적 스케일링: 작은 분자(2개), 큰 분자(5% = 9개 @180원자)

---

### ✅ 원칙 3: 폴백 로직 무결성

**오류 처리 계층:**
1. **PASS** (`error < 1e-4`): 정규화 수행, 예상 값으로 보정
2. **FAIL** (`error > 1e-4`): 정규화 안 함, 원본 데이터 보존
3. **WARNING**: 로그 출력, 사용자 경고

**데이터 무결성:**
- ✅ 큰 오차 발생 시 원본 데이터 보존 (디버깅 용)
- ✅ Silent failure 없음
- ✅ 모든 검증 단계 로그 출력
