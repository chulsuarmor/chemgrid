## 3. 세부 분석 (판단 근거)

### 3.1 그리드 정밀도 및 줌 인터랙션
마우스 휠 줌 인 기능을 통해 그리드가 충분히 확장된 상태에서 테스트를 진행함. 모든 원자가 0.01단위의 정밀도로 그리드 포인트에 정확히 안착(Snap)되었으며, 이로 인해 벤젠과 같은 고리 구조가 완벽한 대칭을 유지함.

### 3.2 툴바 및 전하 표시
`canvas.py` 수정을 통해 양전하(+)와 음전하(-)가 원소 기호의 우측 상단에 작은 위첨자로 가독성 높게 표시됨을 확인. Nitro(N+)와 Oxygen(O-) 배치 시 형전하 보존 상태가 양호함.

### 3.3 화살표 및 텍스트 시각화
이전 보고서에서 지적된 "화살표 머리 미표시" 이슈를 해결함(`drawPolygon` 채움 로직 적용). 텍스트 상자에서 언더바(`_`)를 활용한 화학식 입력 시 실시간으로 아래첨자가 렌더링되어 전문가용 드로잉 툴 수준의 가독성을 확보함.
