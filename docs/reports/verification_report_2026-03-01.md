## 3. 금일 수행한 직접 검증 (2026-03-01 02:00~02:47)

### 3.1 발견 및 수정한 문제

| 문제 | 원인 | 수정 내용 |
|------|------|----------|
| toolbar_setup.py가 적용 안 됨 | **draw.py에 자체 MainWindow + 인라인 툴바 코드** (122줄)가 있어 toolbar_setup.py를 무시 | draw.py의 인라인 코드를 `setup_toolbars(self)` 호출로 교체 (_patch_draw.py) |
| .pyc 캐시로 인한 코드 미반영 | Python이 이전 .pyc를 계속 로드 | 프로세스 종료 후 __pycache__ 전체 삭제 |
| toolbar_setup.py import 누락 | draw.py에 `from toolbar_setup import setup_toolbars` 없음 | _patch_draw.py에서 자동 삽입 |
