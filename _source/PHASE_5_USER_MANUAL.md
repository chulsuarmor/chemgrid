# ChemDraw Pro v1.52 - 최종 사용자 매뉴얼

## 🎯 개요

ChemDraw Pro는 화학 구조 그리기 및 분석을 위한 통합 소프트웨어입니다.
Phase 5에서는 다음 기능들이 추가되었습니다:
- 분자 비교 (Tanimoto 유사도)
- 계산 히스토리 브라우저
- 배치 처리 (다중 분자)
- 올가미 선택 (Lasso Select)

## 📋 시스템 요구사항

| 항목 | 요구사항 |
|------|---------|
| OS | Windows 10/11 |
| Python | 3.10+ |
| PyQt6 | 6.0+ |
| RDKit | 2023.03+ |
| RAM | 2GB 이상 |
| Disk | 500MB 이상 |

## 🚀 시작 가이드

### 1. 애플리케이션 실행
```bash
python draw.py
```

### 2. 분자 그리기
1. 도구 모음에서 "Bond" (기본값) 선택
2. 캔버스에 클릭하여 원자 추가
3. 드래그하여 결합 생성
4. "원소 선택"으로 원소 변경

### 3. 뷰 전환
- **루이스 구조**: 전자 구름과 공명 구조
- **이론적 구조**: 양자 계산 기반 기하학
- **입체 구조**: 3D 분자 구조 (Theory 레이어에서 "입체 구조" 버튼)

## 🔧 새로운 기능 사용법

### A. 분자 비교 (분자 비교 버튼)

#### 목적
두 분자의 유사도를 계산하고 구조적 차이를 분석합니다.

#### 사용 절차
1. 캔버스에 첫 번째 분자 그리기
2. 도구 모음의 **"분자 비교"** 버튼 클릭
3. "분자 2 (비교 대상)" 입력 필드에 다음 중 하나 입력:
   - SMILES: `CC(C)C` (이소부탄)
   - 분자식: `C4H10` (부탄)
4. **"비교 수행"** 버튼 클릭
5. 결과 확인:
   - **요약 탭**: 유사도(%), 동일 분자 여부
   - **상세 정보 탭**: SMILES, 차이점

#### 예제
```
분자 1 (캔버스): 벤젠 (C1=CC=CC=C1)
분자 2 입력: CC1=CC=CC=C1 (톨루엔)
유사도 결과: ~85% (매우 유사한 방향족 화합물)
```

### B. 계산 히스토리 브라우저 (계산 히스토리 버튼)

#### 목적
과거 ORCA 계산 결과를 저장하고 검색합니다.

#### 사용 절차
1. 도구 모음의 **"계산 히스토리"** 버튼 클릭
2. 계산 기록 테이블 확인:
   - ID: 항목 번호
   - 날짜: ISO 형식 타임스탬프
   - 분자식: 분자 조성 (e.g., C6H6)
   - 방법: 계산 방법 (e.g., B3LYP)
   - 에너지: 전자 에너지 (하트리)
   - 상태: converged/failed
   - 시간: 계산 소요 시간 (초)

3. 검색 기능 사용:
   - 검색 필드에 검색어 입력
   - "검색" 버튼 클릭
   - 필터된 결과 확인

#### 검색 예제
```
검색 키워드: "B3LYP" → B3LYP 방법으로 계산한 모든 항목 표시
검색 키워드: "2026-02-06" → 해당 날짜 계산 항목만 표시
```

4. 상세 정보 보기:
   - 테이블에서 행 선택
   - 하단의 "상세 정보" 패널에 표시:
     - SMILES
     - 분자식, 방법, 기저 집합
     - 에너지, HOMO-LUMO Gap
     - 쌍극자 모멘트, 계산 시간

### C. 배치 처리 (배치 처리 버튼)

#### 목적
여러 분자를 한 번에 처리합니다.

#### 사용 절차
1. 도구 모음의 **"배치 처리"** 버튼 클릭
2. "처리할 분자 목록" 입력 필드에 분자들 입력:
   - 형식: SMILES
   - 각 분자는 새 줄에 입력
   ```
   CC(C)C
   C1=CC=CC=C1
   CCO
   ```

3. 계산 옵션 설정:
   - 방법: 기본값 "B3LYP" (선택적)
   - 기저 집합: 기본값 (선택적)

4. **"시작"** 버튼 클릭
   - 진행률 바 표시
   - 진행률 백분위 표시
   - 처리 중인 분자 목록

5. 완료 후:
   - "처리 결과" 목록에 ✓ 표시
   - 진행률 100% 도달
   - 완료 메시지

#### 취소 기능
- 언제든지 **"취소"** 버튼으로 배치 작업 중지

### D. 올가미 선택 (Lasso Select)

#### 목적
Theory 레이어에서 특정 원자 영역을 자유 형태로 선택합니다.

#### 사용 절차
1. **"이론적 구조"** 버튼으로 Theory 레이어 전환
2. 올가미 선택 버튼 **"올가미 선택"** 활성화 (우측 하단 오렌지)
3. 캔버스에서:
   - 마우스 좌측 버튼으로 드래그
   - 원하는 영역을 자유 형태로 그리기
   - 자동으로 올가미 내부의 원자 선택
4. 올가미 경로:
   - 주황색 실선으로 표시
   - 드래그 완료 시 자동으로 사라짐
   - 선택된 원자: 파란색으로 강조

#### 팁
- 올가미는 최소 2점 이상 필요
- 경로는 실시간으로 그려짐
- 5픽셀 간격으로 최적화됨 (부드러운 곡선)

## 🛠️ 문제 해결

### 1. "분자 비교 모듈을 사용할 수 없습니다" 메시지
- **원인**: molecule_comparator.py 파일 누락
- **해결**: 파일 위치 확인 및 RDKit 설치 확인

### 2. Lasso Select 버튼이 보이지 않음
- **원인**: Drawing 모드에서는 Lasso가 비활성화
- **해결**: "이론적 구조" 버튼으로 Theory 레이어로 전환

### 3. 배치 처리 중 느린 속도
- **원인**: ORCA 계산은 시간 소모적
- **대안**: 계산 방법을 더 빠른 것으로 변경 (예: AM1)

### 4. SMILES 입력 오류
- **원인**: 잘못된 SMILES 문법
- **예**: `CCC` (정상) vs `C1=CC1` (불안정한 3-고리)
- **도움**: SMILES 검증기 사용
